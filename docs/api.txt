O que são serviços?
- uma infraestrutura de Integração entre sistemas

Existem vários tipos de serviços:
- Web Service (SOAP)  --> telefone? conexão sincrona (fica conectado por um tempo limitado)
- RESTfull           --> mensagem? conexão assincrona (lista que pode receber e depois responde)


correio:
ws.correios.com.br  // é tipo Web Service

API
4 metodos principais:
Banco de Dados              Significa   Serviços
C - Create                  Incluir     Post
R - Read / Reach/ Research  Consultar   Get
U - Update                  Alterar     Put
D - Delete                  Excluir     Delete

Testes de API
Podem ser realizados de 4 formas:
- Requisição bruta, direta pelo framework de desenvolvimento
- Requisição pela própria documentação dinâmica (.asmx / swagger)
- Requisição através de uma ferramenta configurável
  Exemplo: Postman e SoapUI (teste rápido, para conhecer)
- Requisição via programação
  Exemplo:  Cypress, Mocha + Chai, Supertest (java)
            REST-assured, RESTsharp
            são bibliotecas.
